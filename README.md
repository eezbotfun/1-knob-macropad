# 1-knob-macropad
This is a wireless knob with a 2.0 inch screen, designed for convenient control of volume, playback progress, and other scenarios requiring precise adjustment. 

It can also interact with eezbotfun 8-key macro keyboard to form combos. For example, rotating the knob can switch between the key layouts of the 8-key keyboard.
